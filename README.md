# TO-DO List

Работа со списком задач без отдельной регистрации, но с файловым хранением.

<img src="https://cdn-icons-png.flaticon.com/512/3082/3082854.png" width="200" align="right" hspace="20">

### Технологии

* Node.js -> Express.js
* fs, path

### Настройка проекта

```
npm install

mkdir files

npm run start
```
